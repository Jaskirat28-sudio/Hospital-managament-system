# Hospital Management System

## Overview
This is a Hospital Management System built using Python, Django, and HTML.  
The project provides basic hospital management features including user authentication, doctor listing, hospital dashboard, and appointment booking.

## Features
- User Login and Registration
- Hospital Dashboard
- Doctor List Display
- Appointment Booking System
- About Section
- Responsive Web Pages using HTML

## Tech Stack
- Python
- Django
- HTML
- CSS

## Project Structure
- `doctor_project/` – Main Django project folder
- `hospital/` – Application logic and views
- `templates/` – HTML templates
- `manage.py` – Django management file
