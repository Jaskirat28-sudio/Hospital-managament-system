About This Project

This Hospital Management System is a web-based application developed using Django. The main objective of this project is to simplify hospital operations such as managing doctors and booking appointments. The system allows users to register, log in, view available doctors, and book appointments based on date and time availability. It also prevents double booking and provides a dashboard to track appointment status.

Technologies Used

  • Programming Languages: Python, C, C++
  • Web Technologies: HTML, CSS
  • Framework: Django
  • Database: SQLite (SQL)
